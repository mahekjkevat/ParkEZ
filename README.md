# ParkEZ – Smart Parking System

I completed my internship at **Elaynetech, Surat** for **3 months** and developed **ParkEZ**, a smart parking management system.

ParkEZ is a parking solution focused on the **customer module** for now. It supports parking discovery, slot viewing, booking flow, payment flow, QR ticket generation, booking history, profile management, and related user actions.

> Note: This repository currently includes only the **Customer** screenshots.  
> Owner and Watchman modules will be added later.

---

## Project Module Description

ParkEZ is a smart parking management platform designed to simplify parking search, booking, payment, and ticket generation.  
The customer app allows users to find parking locations, view parking details, select slots and duration, complete payment, and receive QR-based booking confirmation.  
This project is built as a practical internship project and is organized for future expansion into Owner and Watchman modules.

---

## Customer Module

The Customer module includes:
- Login and registration
- OTP / authentication screens
- Splash and onboarding screens
- Home with map-based parking search
- Parking details and filters
- Slot and date-time selection
- Booking flow
- Payment flow
- E-ticket / QR code screen
- Booking history
- Profile and settings screens
